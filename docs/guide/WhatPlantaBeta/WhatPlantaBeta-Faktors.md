::: details What's with the weird way that <span class="labor">Aktion</span> is spelled?

You will notice that some of the spellings that <span class="beta">BETAGLOT</span> uses are not English, this is because whenever possible I have tried to use names which are composed of a hybrid between Latin and Greek. This naming convention is designed to fit together with the languages of Biology, Physics, Geology, Philosophy and Psychology, which are all based in Latin or Greek.

- _Homo Sapien, Sodium Chloride, Glacier, Psychoanalysis, Bacteria, Biceps Brachii, Flexion, Pandemic, Echo_

The purpose of changing the spelling is to enable people from various languages to be able use the names without translation, to make it easier to share and communicate globally. To ensure that the names are easy to pronounce <span class="beta">BETAGLOT</span> uses the letter "K" instead of "C" when relevant and also removes silent letters, like "H".
